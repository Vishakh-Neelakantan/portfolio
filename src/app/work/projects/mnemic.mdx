## Overview
The Spaced Repetition Study Optimizer is an AI-driven intelligent scheduling system designed to enhance learning efficiency. By leveraging machine learning techniques, it optimizes study schedules based on user performance and retention rates.

## Features
- Developed an AI-based intelligent scheduler that utilizes <strong>spaced repetition</strong> to optimize study sessions, significantly improving learning efficiency.
- Integrated an adaptive scheduling logic in the <strong>FastAPI backend</strong> and a responsive <strong>Next.js frontend</strong> to deliver personalized study insights based on user performance.
