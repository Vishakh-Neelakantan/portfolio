## Overview
The File Integrity Manager System is designed to monitor and verify the integrity of files using cryptographic hash functions. It ensures data security by detecting unauthorized changes in real-time.

## Features
- Uses cryptographic hash functions for integrity verification.
- Real-time detection of unauthorized file modifications.
- Enhances security by preventing undetected tampering.
