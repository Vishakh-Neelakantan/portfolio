## Overview
Sign2Speak is a mobile application that translates sign language gestures into English text using the EfficientNetB0 model, enabling better communication for people with hearing and speech impairments.

## Features
- Machine learning-based sign language recognition.
- Real-time translation into English text.
- Mobile-friendly interface.
